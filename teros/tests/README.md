# Tests for Teros

This directory contains the unit tests for the Teros project. The tests are organized into subdirectories based on the modules they test.

## Subdirectories

### `functions`

This subdirectory contains tests for general utility functions.

- `test_slabs.py`: Tests for the `slabs.py` module.

### `thermodynamics`

This subdirectory contains tests for the thermodynamics-related modules.

- `test_binary.py`: Tests for the `binary.py` module.
- `test_formation.py`: Tests for the `formation.py` module.
- `test_ternary.py`: Tests for the `ternary.py` module.
